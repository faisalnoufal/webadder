Ollama Cheat Sheet

1. Installation

Install Ollama

pip install ollama

Verify Installation

ollama --version

2. Authentication

Set API Key

import os
os.environ['OLLAMA_API_KEY'] = 'your_api_key_here'

Use Google Colab Secrets (Recommended)

from google.colab import userdata
api_key = userdata.get('OLLAMA_API_KEY')

3. Basic Usage

Initialize Ollama

import ollama
client = ollama.Client()

Chat with a Model

response = ollama.chat(
    model="deepseek_model",  # Replace with your model name
    message="Your query here"
)
print(response)

List Available Models

models = ollama.list_models()
print(models)

4. Advanced Usage

Customize Model Parameters

response = ollama.chat(
    model="deepseek_model",
    message="Your query here",
    temperature=0.7,  # Adjust creativity (0 to 1)
    max_tokens=100    # Limit response length
)

Stream Responses

for chunk in ollama.chat(
    model="deepseek_model",
    message="Your query here",
    stream=True
):
    print(chunk['message'], end="")

5. Troubleshooting

Common Errors

  • Missing API Key: Ensure the API key is set correctly. 
    os.environ['OLLAMA_API_KEY'] = 'your_api_key_here'
  • Model Not Found: Verify the model name is correct. 
    models = ollama.list_models()
print(models)

Debugging

import logging
logging.basicConfig(level=logging.DEBUG)

6. Useful Commands

Update Ollama

pip install --upgrade ollama

Uninstall Ollama

pip uninstall ollama

7. Example Workflow

Full Example in Python

import ollama

# Set API Key
import os
os.environ['OLLAMA_API_KEY'] = 'your_api_key_here'

# Initialize Ollama
client = ollama.Client()

# Chat with a model
response = ollama.chat(
    model="deepseek_model",
    message="Explain quantum computing in simple terms."
)

# Print the response
print(response)
